# PathAI - AI-Powered Goal Roadmap Generator

A local-first web application that generates personalized quest-based roadmaps for achieving any goal using AI.

## Features

- 🎯 **AI-Generated Roadmaps**: 100 unique, actionable quests generated by Groq AI
- 🔐 **Local Storage**: All data stored in browser's IndexedDB - no server database needed
- 🗺️ **Fog of War Dashboard**: Current 10 quests visible, remaining locked until tier completion
- 🎮 **Gamification**: Progress tracking, tier unlocks, confetti celebrations
- 👥 **Multi-Goal Support**: Manage multiple goals via Orbit view
- ⚡ **Offline Capable**: Works offline after initial load

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Backend**: Django (for AI API only)
- **AI**: Groq API (Llama 3.3)
- **Storage**: IndexedDB (client-side)

## Quick Start

### Prerequisites
- Python 3.8+
- Groq API key

### Setup

1. Clone the repository:
```bash
git clone <your-repo-url>
cd pathi
```

2. Create virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# or
venv\Scripts\activate  # Windows
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Create `.env` file:
```env
GROQ_API_KEY=your_groq_api_key_here
SECRET_KEY=your_secret_key
```

5. Run migrations (if using SQLite):
```bash
python manage.py migrate
```

6. Start server:
```bash
python manage.py runserver
```

7. Open `http://127.0.0.1:8000`

## Deployment

### Option 1: Render (Recommended)

1. Push code to GitHub:
```bash
git add .
git commit -m "Add deployment config"
git push origin main
```

2. Create a Web Service on [Render](https://render.com):
   - Connect your GitHub repo
   - Build Command: `pip install -r requirements.txt && python manage.py migrate`
   - Start Command: `gunicorn pathai.wsgi --bind 0.0.0.0:$PORT`

3. Add Environment Variables in Render dashboard:
   - `GROQ_API_KEY`: your Groq API key
   - `SECRET_KEY`: generate a secure key
   - `DEBUG`: `False`
   - `DATABASE_URL`: will be auto-created by Render (PostgreSQL)

4. Your app will be live at `https://your-app.onrender.com`

### Option 2: Static Frontend (Recommended for full local storage)

The frontend is entirely client-side and can be deployed to any static hosting:

1. Build static files:
```bash
python manage.py collectstatic
```

2. Deploy the `static/` folder and `templates/index.html` to:
   - Netlify
   - Vercel
   - GitHub Pages
   - Cloudflare Pages

### Option 2: Heroku / Render / Railway

1. Set environment variables:
   - `GROQ_API_KEY`
   - `SECRET_KEY`

2. Deploy Django app

## Usage

1. **Register**: Create account with age, industry, interests
2. **Set Goal**: Enter your goal (e.g., "learn JavaScript")
3. **AI Generation**: Get 100 unique quests divided into 10 tiers
4. **Complete Quests**: Mark quests done, unlock next tier at 10 completions
5. **Progress**: Track overall progress percentage
